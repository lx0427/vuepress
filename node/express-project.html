# 插件用途

插件名 用途
cors (opens new window) 跨域请求
express.json (opens new window) 解析带有 JSON 负载的传入请求
http-assert (opens new window) 接口报错提示
multer (opens new window) 上传图片
express.static (opens new window) 静态文件托管
bcrypt (opens new window) 数据散列比较
jsonwebtoken (opens new window) token 生成解析
inflection (opens new window) 数据大小写驼峰转换
