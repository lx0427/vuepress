# reset.css 与 normalize.css

  • reset 重置元素样式达到统一
  • normalize 模块化
  • normalize 不会出现大段继承
  • normalize 保留许多浏览器默认样式，尽可能让样式与现代标准相符合

# :last-child & :last-of-type

:last-child

  • :last-child: 不能有干扰元素

:last-of-type

  • :last-of-type: 可以有干扰元素
  • :last-of-type: 只作用于标签
  • div.box:last-of-type: 最后一个div且含有类名box
