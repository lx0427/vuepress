# 添加分类配置

  1. 设定好类型，填入分类配置表
  2. 有名称，需要配置hasNameShowArr
  3. 名称有长度限制，，需要配置limitNameArr
  4. 所有有限制的分类需配置judgeLimit(id)
    • this.limitNum = 5 限制名称长度
    • this.limitCount = 4 限制个数
    • this.limitImage = true 无图
