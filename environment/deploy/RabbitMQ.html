# RabbitMQ

  1. 下载并安装 erlang (opens new window), 选择OTP 23.0 Windows 64-bit Binary File
  2. 配置环境变量ERLANG_HOME: D:\Program Files\erl-23.0
  3. 系统变量 path 中新增：%ERLANG_HOME%\bin
  4. cmd：erl出现版本号
  5. 下载并安装 RabbitMQ (opens new window)，选择Windows installer
  6. 切到安装目录的 D:\Program Files\RabbitMQ Server\rabbitmq_server-3.8.7\sbin
    rabbitmq-plugins enable rabbitmq_management
    1
  7. 访问http://localhost:15672, guest/guest登录，即可
