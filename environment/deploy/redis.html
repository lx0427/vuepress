下载地址 (opens new window), 选择 zip 解压

# 启动 redis 服务

redis 解压目录, 打开 cmd

redis-server.exe redis.windows.conf

# 启动端口进程

启动 redis 服务, redis 解压目录, 打开 cmd

redis-cli.exe -h 127.0.0.1 -p 6379

redis-cli.exe -h 127.0.0.1 -p 6379 -a 123456

    auth 123456
