## 园区物流

> 原辅料管理

计算逻辑

```md
- 地磅调整数(默认为 0)
- 特殊调整数(默认为 0)

1. 发货净重 = 发货毛重 - 发货皮重
2. 收货净重 = 收货毛重 - 收货皮重
3. 实际收货数量 = 收货净重 + 地磅调整数 + 特殊调整数
4. 差额 = 实际收货数量 - 发货净重
5. 发货平均重量/包 = 发货净重 / 袋数
6. 收货平均重量/包 = 实际收货数量 / 袋数
```
