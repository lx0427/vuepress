# 福利报名

::: tip
福利列表信息：后台(homeworkstation)/文章列表/福利
:::

## 后台插入报名类型
| 类型     | 对应值 |
| -------- | ------ |
| 看房团   | 7      |
| 独立报名 | x      |

## 报名状态+报名人数
| 对应值   | 状态       |
| -------- | ---------- |
| applyNum | 已报名人数 |
| status:0 | 报名中     |
| status:1 | 报名结束     |

报名结束的情况
* 看房团商品被删除
* 发布状态为未发布
* 看房时间过期
