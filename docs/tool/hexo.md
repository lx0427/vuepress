# hexo

## 安装使用

```
npm install -g hexo-cli
hexo new first

hexo init
npm install
npm install hexo --save

git clone https://github.com/iissnan/hexo-theme-next themes/next

* theme: next

npm install hexo-deployer-git -save

git init

git remote add origin https://github.com/lx0427/lx0427.github.io.git

hexo clean && hexo g && hexo d

git add

git commit -m 'init'

git push origin

```
