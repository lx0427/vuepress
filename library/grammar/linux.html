# ls

查看目录列表信息

# 列出文件属性, 缩写为ll
ls -l
# 加h，自动变换单位
ls -lh
# 列出文件目录
ls -d
# 列出文件目录的属性
ls -ld
# 查看目录所有文件（包括隐藏文件（带一个点 .））
ls -la
# 加t，文件按时间排序，越早越排在下面
ls -lta

# rm

  1. rm -r -r 表示删除的是目录
  2. rm -f -f 表示免确认删除文件夹下面的文件和文件夹
  3. rm -rf 免确认删除文件夹(含目录中的所有文件及文件夹)
